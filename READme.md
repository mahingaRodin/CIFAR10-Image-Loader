# CIFAR10 Image Loader and Visualizer

This project loads and visualizes the CIFAR-10 dataset using PyTorch.

## Features:
- Downloads the CIFAR-10 dataset.
- Displays a batch of images with their corresponding labels.

## Requirements:
- Python 3.x
- PyTorch
- torchvision
- matplotlib
- numpy
